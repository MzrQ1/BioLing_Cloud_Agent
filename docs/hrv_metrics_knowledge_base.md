# HRV 指标知识条目

## SDNN（NN间期标准差）
- 定义：所有正常R-R间期的标准差，反映总体自主神经调节能力
- 正常范围：健康成人 141±39 ms（24h）；短期(5min) 50±16 ms
- 临床意义：
  - SDNN < 50ms（24h）：严重自主神经功能受损，心脏性猝死风险增加
  - SDNN < 100ms（24h）：中等风险
  - 压力下典型下降幅度：15-30%

## RMSSD（相邻NN间期差值均方根）
- 定义：相邻R-R间期差值的均方根，主要反映迷走神经（副交感）活动
- 正常范围：健康青年 42±15 ms（5min）
- 临床意义：
  - 高RMSSD → 良好的副交感神经张力 → 放松/恢复状态
  - 低RMSSD → 副交感抑制 → 压力/疲劳/焦虑状态
  - 对呼吸调控敏感，深呼吸时显著升高

## LF/HF 比值
- 定义：低频功率(0.04-0.15Hz)与高频功率(0.15-0.4Hz)的比值
- 解读（需谨慎）：
  - 传统观点：反映交感/副交感平衡（已被质疑）
  - 现代观点：LF受呼吸、压力反射等多因素影响，
    不能简单等同于交感活动
  - 共振呼吸(~0.1Hz)时LF显著增大，是正常生理反应而非压力指标

## pNN50
- 定义：相邻NN间期差值>50ms的比例
- 意义：副交感活动的另一指标，与RMSSD高度相关
- 压力状态下典型值：<5%；放松状态：>20%
