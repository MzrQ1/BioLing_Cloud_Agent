# 呼吸调节技术知识库

## 技术1：膈式呼吸（Diaphragmatic Breathing）
- 方法：吸气时腹部膨胀，呼气时腹部收缩
- 频率：6-10次/分
- 疗效证据：
  - 降低皮质醇水平（d = 0.45）
  - 降低焦虑（d = 0.52）
  - 改善HRV
- 适用场景：日常压力管理、焦虑缓解

## 技术2：4-7-8呼吸法
- 方法：吸气4秒→屏息7秒→呼气8秒
- 机制：延长呼气相 → 增强迷走刺激 → 副交感激活
- 适用场景：入睡困难、急性焦虑
- 注意：对初学者可能过于困难，建议从4-4-6开始适应

## 技术3：箱式呼吸（Box Breathing）
- 方法：吸气4秒→屏息4秒→呼气4秒→屏息4秒
- 适用场景：高压环境下快速镇静（军事、急救人员常用）
- 疗效证据：降低急性应激反应

## 技术4：生理叹息（Physiological Sigh）
- 方法：连续两次短吸（双吸）+ 一次长呼
- 机制：双吸重新扩张肺泡 → 增加CO2排出效率 → 快速镇静
- 疗效证据：Stanford研究表明5分钟生理叹息练习效果
  优于等时长冥想和HRV生物反馈（Balban et al., 2023）
- 适用场景：需要最快速的镇静效果（1-3分钟）

## 智能体选择逻辑建议
| 用户状态 | 推荐技术 | 理由 |
|----------|----------|------|
| 轻度紧张(HR+5-10, HRV轻度↓) | 膈式呼吸 6bpm | 温和有效 |
| 中度焦虑(HR+10-20, HRV明显下降) | 箱式呼吸 或 共振呼吸 | 结构化引导 |
| 急性应激(HR+20, SCR明显上升) | 生理叹息 | 最快起效 |
| 入睡困难(HR正常, 思绪杂乱) | 4-7-8呼吸 | 延长呼气促进入睡 |
