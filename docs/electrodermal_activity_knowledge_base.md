# 皮肤电活动知识条目

## 核心概念
- 皮肤电导(SC)由汗腺活动驱动，受交感神经系统单独控制
  （无副交感支配）
- 两个成分：
  - 皮肤电导水平(SCL/Tonic)：基线水平，反映总体唤醒
  - 皮肤电导反应(SCR/Phasic)：事件相关的快速波动

## SCR 指标与唤醒
- SCR频率：非特异性SCR频率 > 20次/5min → 高唤醒状态
- SCR幅度：> 0.05 μS 为有效SCR（阈值标准）
- 潜伏期：刺激后1-5秒出现
- 恢复半时间：SCR达峰后恢复50%所需时间，越短=越高唤醒

## 压力状态下的EDA变化
- 急性压力：SCR频率增加300-500%，SCL上升2-5 μS
- 放松状态：SCR频率 < 3次/5min，SCL逐渐下降
- 深呼吸时：SCL在2-3分钟后开始下降，SCR频率减少

## 运动伪影注意事项
- 手指运动可产生伪SCR（振幅可达0.1-0.5 μS）
- 识别方法：运动伪影的上升时间通常<0.5s（正常SCR为1-3s）
- 建议：静态测量场景中伪影风险较低
