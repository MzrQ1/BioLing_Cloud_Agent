# HRV-压力-情绪 循证知识

## 急性压力对HRV的影响
- 机制：压力激活下丘脑-垂体-肾上腺(HPA)轴 → 皮质醇升高 → 
  交感激活 → 迷走神经抑制 → HRV降低
- 效应量：急性心理压力下SDNN平均降低20-30%
- 恢复时间：轻度压力后15-30min可恢复；重度压力需数小时

## 慢性压力对HRV的影响
- 长期低HRV与以下状况相关：
  - 慢性工作压力（r = -0.31）
  - 焦虑障碍（d = 0.45-0.89）
  - 抑郁症（d = 0.34-0.68）
  - 职业倦怠（r = -0.25）

## HRV与情绪调节能力
- 神经内脏整合模型(Neurovisceral Integration Model)：
  高静息HRV → 更好的前额叶-杏仁核连接 → 更灵活的情绪调节
- 高HRV个体在情绪调节任务中表现更好（d = 0.56）
