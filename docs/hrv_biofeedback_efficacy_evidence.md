# HRV 生物反馈疗效证据

## 共振频率呼吸
- 定义：以个体共振频率（通常4.5-6.5次/分）进行呼吸，
  最大化呼吸性窦性心律不齐(RSA)
- 即时效果：
  - SDNN提升30-50%（单次训练）
  - RMSSD提升20-40%
  - LF功率显著增加（RSA共振效应）
- 长期效果（4-10周训练）：
  - 焦虑降低（d = 0.81）
  - 抑郁降低（d = 0.58）
  - 静息HRV基线提升


## 最佳训练参数
- 呼吸频率：6次/分（10秒周期）为群体最优近似值
- 吸呼比：建议1:2（吸4秒呼6秒）以最大化迷走刺激
- 单次有效时长：5-20分钟
- 个体共振频率测定：需通过频率扫描(4-7次/分)找到
  RSA振幅最大的频率
